# Bard Project - Claude Integration

## Project Overview
This is a React + Vite token exchange application with Supabase integration.

## Tech Stack
- **Frontend**: React 18, Vite
- **Styling**: Tailwind CSS
- **Backend**: Supabase
- **Icons**: Lucide React

## Development Commands
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## Project Structure
```
bard-main/
├── src/
│   ├── app.jsx          # Main app component (minimal)
│   ├── main.jsx         # Entry point and main logic
│   ├── WalletInput.jsx  # Wallet input component
│   └── index.css        # Global styles
├── public/
│   └── _redirects       # Netlify redirects
├── package.json
├── vite.config.js
├── tailwind.config.js
└── netlify.toml
```

## Key Features
- Token exchange functionality
- Wallet integration
- Supabase database connection
- Responsive design with Tailwind
